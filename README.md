# QR Code Generator

A single-file browser app for generating styled QR codes — no build step, no dependencies to install.

## Features

- **URL mode** — generate a QR code from any link
- **WiFi mode** — generate a scannable WiFi credential QR code (supports WPA/WPA2, WEP, or open networks)
- **Styling** — choose dot shape, corner shape, foreground/background colors
- **Logo overlay** — embed an image in the center with adjustable size
- **Download** — export as PNG or SVG

## Usage

Open `index.html` in any modern browser. No server required.

## Dependencies

- [qr-code-styling](https://github.com/kozakdenys/qr-code-styling) v1.6.0-rc.1 (loaded via CDN)
