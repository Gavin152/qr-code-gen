Add localStorage persistence to app.js. All new code is added to the existing file — no new files created.

**1. Storage key and helper functions (add near top of file, after the global variable declarations on lines 1-2):**

Use a single localStorage key `qr-settings` storing a JSON object. Create two helper functions:

```javascript
function saveSettings(settings) {
  try { localStorage.setItem('qr-settings', JSON.stringify(settings)); } catch (e) { /* silent — per D-07 */ }
}

function loadSettings() {
  try { const s = localStorage.getItem('qr-settings'); return s ? JSON.parse(s) : null; } catch (e) { return null; }
}
```

These wrap all localStorage access in try/catch so the app works normally if localStorage is unavailable (private browsing, quota exceeded) — per D-07.

**2. Create a `saveAllSettings()` function (add after the helpers):**

This function reads the current state of all saveable settings from the DOM and globals, then calls `saveSettings()`. It saves:

- `activeTab` — from the global variable (string: 'url' or 'wifi')
- `fgColor` — from `document.getElementById('fg-color').value`
- `bgColor` — from `document.getElementById('bg-color').value`
- `dotStyle` — from `document.getElementById('dot-style').value`
- `cornerStyle` — from `document.getElementById('corner-style').value`
- `logoSize` — from `document.getElementById('logo-size').value` (string, e.g., "30")
- `logoDataUrl` — from the global variable, but ONLY if `logoDataUrl !== null && logoDataUrl.length <= 500000` (per D-03, skip saving logo image if over ~500KB, save null instead)
- `logoFileName` — from `document.getElementById('logo-name').textContent` (for restore display)
- `downloadFormat` — from the global variable (per D-04, lightweight to include)
- `urlInput` — from `document.getElementById('url-input').value`
- `wifiSsid` — from `document.getElementById('wifi-ssid').value`
- `wifiSecurity` — from `document.getElementById('wifi-security').value`
- Do NOT save `wifi-password` — per D-02 (security concern)

**3. Hook `saveAllSettings()` into the `update()` function:**

Add `saveAllSettings();` as the LAST line inside the existing `update()` function body (after the `qrCode.update(...)` call on line 45). Since every user interaction already flows through `update()`, this single hook captures changes to: fg color, bg color, dot style, corner style, logo size, URL input, WiFi SSID, WiFi security.

**4. Hook save into `setFormat()` (lines 101-105):**

Add `saveAllSettings();` as the last line in `setFormat()`. This is needed because `setFormat()` does NOT call `update()`, so the save hook inside `update()` won't fire when the user changes download format.

Final save hooks (only two insertion points):
- Inside `update()` — covers all appearance settings, content fields, active tab, logo
- Inside `setFormat()` — covers download format

Add a `DOMContentLoaded` event listener that restores all saved settings from localStorage. Add this at the END of app.js (after all existing code, including the event listeners added in Task 1).

Note: `app.js` is loaded via a synchronous `<script>` at end of `<body>`. The `DOMContentLoaded` event has NOT yet fired when this code runs (it fires after all synchronous scripts complete), so the listener will be called.

```javascript
document.addEventListener('DOMContentLoaded', () => {
  const saved = loadSettings();
  if (!saved) return;

  // Restore active tab (D-05, D-06, PERS-04)
  if (saved.activeTab && saved.activeTab !== activeTab) {
    activeTab = saved.activeTab;
    document.querySelectorAll('.tab').forEach(t => {
      t.classList.toggle('active', t.dataset.tab === activeTab);
    });
    document.querySelectorAll('.panel').forEach(p => p.classList.remove('active'));
    document.getElementById('panel-' + activeTab).classList.add('active');
  }

  // Restore colors (PERS-01)
  if (saved.fgColor) {
    document.getElementById('fg-color').value = saved.fgColor;
    document.getElementById('fg-hex').textContent = saved.fgColor;
  }
  if (saved.bgColor) {
    document.getElementById('bg-color').value = saved.bgColor;
    document.getElementById('bg-hex').textContent = saved.bgColor;
  }

  // Restore dot and corner styles (PERS-01)
  if (saved.dotStyle) document.getElementById('dot-style').value = saved.dotStyle;
  if (saved.cornerStyle) document.getElementById('corner-style').value = saved.cornerStyle;

  // Restore logo (D-03, D-05)
  if (saved.logoDataUrl) {
    logoDataUrl = saved.logoDataUrl;
    document.getElementById('logo-name').textContent = saved.logoFileName || 'logo.png';
    document.getElementById('logo-clear').style.display = 'inline';
    document.getElementById('logo-size-field').style.display = 'block';
  }

  // Restore logo size (PERS-02)
  if (saved.logoSize) {
    document.getElementById('logo-size').value = saved.logoSize;
    document.getElementById('logo-size-val').textContent = saved.logoSize + '%';
  }

  // Restore download format (D-04)
  if (saved.downloadFormat) {
    downloadFormat = saved.downloadFormat;
    document.getElementById('fmt-png').classList.toggle('active', downloadFormat === 'png');
    document.getElementById('fmt-svg').classList.toggle('active', downloadFormat === 'svg');
  }

  // Restore content fields (D-02)
  if (saved.urlInput) document.getElementById('url-input').value = saved.urlInput;
  if (saved.wifiSsid) document.getElementById('wifi-ssid').value = saved.wifiSsid;
  if (saved.wifiSecurity) document.getElementById('wifi-security').value = saved.wifiSecurity;
  // wifi-password deliberately NOT restored (D-02)

  // Trigger QR code update to reflect restored settings (D-05)
  update();
});
```

Key implementation details:
- The `DOMContentLoaded` listener goes at the END of app.js so all globals, functions, and the QRCodeStyling instance are already initialized.
- Tab restore uses the same toggle logic as the existing tab click handler (toggle `.active` class on `.tab` buttons and `.panel` divs) — per D-06.
- Logo restore shows `#logo-name`, `#logo-clear`, and `#logo-size-field` to match the state the upload handler creates — per D-05.
- Logo filename falls back to `'logo.png'` if `logoFileName` was not saved — per D-05 (CONTEXT.md interaction contract 3).
- `update()` is called LAST so the QR canvas immediately reflects all restored settings — per D-05.
- The `update()` call at the end will also trigger `saveAllSettings()` (from Task 1), which is harmless — it re-saves the same values just loaded.
