<layout>
## Desktop (>=768px)
.app is a CSS grid: `grid-template-columns: 1fr 1fr; grid-template-areas: "header header" "form preview";`
- .app-header spans full width
- .form-col is the left column (tabs, panels, style controls)
- .preview-col is the right column (QR canvas + download row), sticky top: 2rem

## Mobile (<768px)
.app is single column, stacked: header → form → preview → download

## Social Sidebar (desktop >= 768px)
position: fixed; right: 1rem; top: 50%; transform: translateY(-50%)
Flex column, gap: 0.75rem
Two icon links (Ko-fi heart, GitHub mark) — 40x40px click targets, icon size 20px

## Social Footer (mobile < 768px)
position: fixed; bottom: 0; left: 0; right: 0; height: 48px
Flex row, centered, gap: 2rem
Same two icon links with small text label beside each icon
Body gets padding-bottom: 56px on mobile to prevent content being hidden behind footer

## Body padding
Desktop: padding-right: 56px to prevent content overlap with sidebar
</layout>
